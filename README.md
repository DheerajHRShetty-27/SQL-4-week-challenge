## üìò Case Study 1: Danny's Diner

**Objective:** Analyze customer visiting patterns, spending behavior, and menu performance for a small Japanese-themed diner.  

**Key Insights:**
- Identified customer loyalty trends using visit frequency and total spending.
- Analyzed most frequently ordered menu items and high-value customers.
- Used SQL to compute first-time purchase, repeat visits, and favorite dish for each customer.
- Helped understand how Danny can improve customer retention through menu optimization.

**SQL Concepts Used:**  
`JOIN`, `GROUP BY`, `ORDER BY`, `Aggregate Functions`, `Date-based filtering`
